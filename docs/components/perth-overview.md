# Perth: Your Terminal, Multiplied

## What It Does

Perth is a custom distribution of Zellij, a terminal multiplexer that transforms your single terminal window into a sophisticated workspace with multiple panes, tabs, and sessions. Think of it as converting a tiny studio apartment into a multi-room mansion, all within your terminal.

## Why It Matters

Developers live in the terminal. You're running a development server in one window, tailing logs in another, running tests in a third, and executing Git commands in a fourth. Without a multiplexer, you're either constantly creating new terminal windows or awkwardly switching between tabs, losing your mental map of where everything is.

Perth solves this by letting you organize your terminal workspace exactly how you think. Create layouts with multiple panes visible simultaneously. Group related tasks into tabs. Save entire session configurations so you can resume complex setups instantly.

**For Team Collaboration**: Share terminal session configurations. Junior developers can load the exact same workspace layout as senior developers, reducing onboarding friction and ensuring consistent development environments.

## The Perth Difference

While Perth is built on Zellij's solid foundation, it's customized for the 33GOD ecosystem with sensible defaults, integrated keybindings, and seamless coordination with other workspace management tools like iMi.

Perth eliminates terminal chaos. It's the difference between juggling windows and having a thoughtfully organized workspace where everything has its place. Perfect for backend developers, DevOps engineers, and anyone who needs to monitor multiple processes simultaneously.
